# Unit 0 Final Practice

This repository contains example code for my dream application, **StudyBuddy**.

## Dream Application Description
StudyBuddy is an app for students who want help staying organized and motivated with schoolwork. It would let users track assignments, monitor progress, save reminders, and get encouraging feedback as they complete tasks.

## Features
- Assignment tracking
- Progress percentage
- Reminder messages
- Study subjects list
- Student profile information
